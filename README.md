Dashboard interactif de suivi des commandes de production.  
**100% navigateur** — aucun serveur, aucune installation.

## 🌐 Accès

👉 **https://TON-PSEUDO.github.io/planning-production/**

*(Remplace TON-PSEUDO par ton nom d'utilisateur GitHub)*

## 🚀 Mise en place (5 minutes)

1. **Créer le dépôt** : Cliquez sur "Use this template" ou créez un nouveau repo
2. **Activer GitHub Pages** : Settings → Pages → Source: `main` → `/ (root)` → Save
3. **Attendre 2 minutes** : GitHub génère le site
4. **Ouvrir le lien** : `https://votre-pseudo.github.io/planning-production/`

## 📊 Utilisation

1. **Premier accès** : Glissez votre fichier Excel (.xlsx) sur la page
2. **Le dashboard s'affiche** avec toutes vos commandes
3. **Remplissez le suivi** : commercial, demande, réponse, version, plans, nomenclature
4. **Les données sont sauvegardées** automatiquement dans votre navigateur

## 📤📥 Partager le suivi entre collègues

Le suivi est stocké dans le navigateur de chaque utilisateur. Pour synchroniser :

1. **Exporter** : Cliquez `📤 Exporter suivi` → télécharge un fichier JSON
2. **Partager** : Envoyez ce JSON à vos collègues (email, réseau partagé...)
3. **Importer** : Ils cliquent `📥 Importer suivi` → sélectionnent le JSON

> 💡 **Astuce** : Placez le JSON sur un dossier réseau partagé. Chacun importe la dernière version.
